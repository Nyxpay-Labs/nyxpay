## [1.0.0] - 2024-01-01

### Added

- Initial release of NyxPay protocol
- Solana program for on-chain proof verification and state transitions
- Zero-knowledge proof system for private transactions
- Stealth address generation and management
- Privacy score computation
- Selective disclosure proofs
- Wallet architecture and documentation
- Security documentation and threat model
- Comprehensive protocol documentation

### Security

- Zero-knowledge proofs for transaction privacy
- Stealth addresses for recipient privacy
- Balance preservation enforcement
- Double-spend prevention
- Non-negative transfer enforcement

### Documentation

- Protocol overview and architecture
- Security model and threat analysis
- Zero-knowledge proof specifications
- Wallet architecture and usage
- Contributing guidelines
- Code of conduct

## Change Classification

Changes are classified as follows:

- **Added**: New features
- **Changed**: Changes to existing functionality
- **Deprecated**: Features that will be removed in a future release
- **Removed**: Removed features
- **Fixed**: Bug fixes

## Versioning

Versions follow semantic versioning (MAJOR.MINOR.PATCH):

- **MAJOR**: Incompatible API or protocol changes
- **MINOR**: New functionality in a backward-compatible manner
- **PATCH**: Backward-compatible bug fixes
